# Self-collaboration Code Generation via ChatGPT

基于 Yihong Dong 等人 [**Self-collaboration Code Generation via ChatGPT**](https://arxiv.org/abs/2304.07590)

开发的面向事件风暴建模的自我协作框架

## Run code
```bash
# generate
bash run.sh
# evaluate
bash evaluate.sh
```
